# 鯖江市図書館マップ「さばとマップ」

### 対象プラットフォーム

- iOS (iOS8.3+)
- Android (4.4+)
- Windows (10+)

### ビルド環境の整備

- node.js (0.12+)
- bower
- Cordava 5.1.1

```bash
npm install -g bower
npm install -g cordova
npm update
bower update
gulp
cordova platform update ios
cordova platform update android
cordova platform update windows
```
